[1] For tests involving a tcp connection from one emulator instance to another one, a bit of setup ("port redirection") is required. Pick two distinct port numbers for the applications to use: <port1> and <port2>. Telnet into the emulated server device by doing

telnet localhost 5554

(the port may vary -- it's the number that "adb devices" reports) and then type

redir add tcp:<port1>:<port2>

Now the client can use "10.0.2.2:<port1>" to connect to the server who is listening on <port2>. Note that you can also connect from your host machine to the emulated server device by doing "telnet localhost <port1>". More info about emulator networking here.
